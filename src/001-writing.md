# 写作

## Markdown 写作工具

## Markdown 语法
- 标题
- 列表
- 超链接
- 任务列表
- 段落与换行
- 列表与段落
- 文字强调
    - 加粗
    - 斜体
    - 删除线
- 分割线
- 引用
- 展示代码
- 表格
- 图片
- 目录



## Markdown 规范

- 每个章节名称 [1-9]_文档名称
- 每个文档对应一个章节，每个章节从标题一开始
- 每行60个汉字，120个英文字符
